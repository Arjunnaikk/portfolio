# cotask - collaborative task management

<br />

cotask is a full-stack task management app built for teams that need to plan and track work together in real time. users create workspaces, assign tasks, set due dates, and see updates from teammates land instantly without refreshing the page.

authentication is handled with nextauth.js so teams can sign in securely and manage access per workspace. all task data is persisted in a d1 sql database, and the frontend uses tanstack query to keep the ui in sync with the server, caching reads and rolling back optimistic updates if a write fails. the result is a responsive interface where multiple people can edit the same board at once without stepping on each other's changes.

## tech stack

- **frontend:** next.js, tailwind css, tanstack query
- **auth:** nextauth.js
- **database:** cloudflare d1 (sql)
- **languages:** typescript
