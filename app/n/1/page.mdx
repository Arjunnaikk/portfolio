meblog is a full-stack web3 blogging platform built on solana, where every post is published and owned on-chain. writers connect a wallet, publish through a rust-based anchor smart contract, and get censorship-resistant, verifiable ownership over what they write, no centralized server can take a post down or reassign it.

each author's posts live under a program derived address (pda), generated deterministically so the chain can locate and validate content without a central registry. the smart contract enforces the platform's business rules directly at the protocol level, and the frontend uses tanstack query to fetch, cache, and keep on-chain state in sync in real time, with solana wallet adapters handling transaction signing.

## tech stack

- **frontend:** next.js, tanstack query
- **blockchain:** solana, anchor, rust smart contracts
- **wallets:** solana wallet adapter
- **languages:** typescript, rust
