# pesca - dsa learning & visualisation platform

<br />

pesca is a unified learning ecosystem for data structures and algorithms, built to replace static tutorials with something you can actually watch run. students step through algorithms in an interactive visualizer and write, test, and run code side by side in a built-in multi-language ide.

the visualization module is animated with gsap, walking through each step, comparison, and pointer movement so the logic is visible rather than abstract. beyond visualizing, pesca includes an assessment module that auto-scores mcqs, tracks individual student progress, and gives teachers an administration hub for performance insights. the backend runs entirely on cloudflare workers with a d1 (sqlite) database, keeping the whole platform serverless and low-latency.

## tech stack

- **frontend:** next.js, gsap
- **backend:** cloudflare workers
- **database:** cloudflare d1 (sqlite)
- **languages:** typescript
